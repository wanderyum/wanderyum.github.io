C++学习笔记(输入输出，文件，字符串)

26 February, 2020

输入输出

需要头文件:

#include<iostream>

输出

std::cout << "输出中文" << std::endl;
double pi = 3.1415926;

字符串

需要头文件:

#include<string>

创建字符串:

std::string str1;
std::string str2 = "string2";
std::string str3("string3");

与字符数组相互转换

char*或char[]转string 直接赋值:

std::string str;
char *cp = "char*";     // 这个写法会报警告
char arr[] = "char[]";
str = cp;
str = arr;

string转char*或char[]

  1. 调用string的data方法
std::string str3="string3";
/* 方案1 */ 
const char *cp1=str3.data();
/* 方案2 */
char *cp2=(char*)str3.data();

这里需要注意方案1中const一定要加, 否则会报错invalid conversion from 'const char*' to 'char*'。
2. 调用string的c_str方法

std::string str4="string4"; 
/* 方案1 */
const char *cp3=str4.c_str();
/* 方案2 */
char *cp4=(char*)str4.c_str();
  1. 调用string的copy()方法
std::string str5 = "string5";
    char c_arr[50];
    str5.copy(c_arr, str5.length(), 0);     // 这里str5.length()代表复制几个字符，0代表复制的位置
    *(c_arr+str5.length())='\0';            // 注意手动加结束符！
    std::cout << c_arr << "\n";
